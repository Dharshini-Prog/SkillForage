# Memory Module

This directory is reserved for persistent memory used by the SkillForge multi-agent system.

## Purpose

The memory module is intended to support future enhancements such as:

- Conversation history
- Agent context sharing
- User preferences
- Cached Skill Blueprints
- Intermediate workflow states

## Current Status

The current version of SkillForge performs stateless execution and does not persist memory between runs.

This folder is included to support future scalability.
